## Guide Content Structure

Each guide includes the following sections:

### 1. Navigation
- How to access the feature in the UI
- Sidebar location
- Quick access shortcuts

### 2. How-To Instructions
Step-by-step instructions for common tasks:
- Creating items
- Editing/updating
- Submitting for approval
- Viewing and filtering
- Exporting data

### 3. Field Reference
Detailed information about each field:
- Field name and label
- Data type
- Required/optional
- Validation rules
- Tips and guidance

### 4. Workflow Information
- Status transitions
- Who can take each action
- Visual workflow diagrams

### 5. Permissions Matrix
- What each role can do
- Restrictions and limitations

### 6. FAQ
Common questions and answers about the feature.

### 7. Related Guides
Cross-references to related features.

Users can ask the AI Assistant how-to questions in natural language:

**General questions:**
- "How do I create a timesheet?"
- "How do I submit an expense claim?"
- "How does approval work?"

**Specific questions:**
- "What fields are required for an expense?"
- "Who can sign a milestone certificate?"
- "What's the difference between Validate and Approve?"

**Troubleshooting:**
- "Why can't I edit my timesheet?"
- "Why can't I see the Finance section?"
- "What happens if my expense is rejected?"

## Maintenance

### Adding New Guides

1. Create guide file in appropriate category folder
2. Add entry to `guideRegistry` in `index.js`
3. Add keywords to `keywordMapping` in `index.js`
4. Update this document

### Updating Existing Guides

1. Edit the guide file directly
2. No registry changes needed
3. Changes take effect immediately
